## Basic TODO list app

### Install

- Install the package using

```
go install github.com/fahadqazi/go-todo/cmd/todo@latest
```

### Usage

- Help

```
todo -h
```

- Add todo

```
todo -add <your-todo>
```

- Clear all todos

```
todo -clear
```

- List all todos

```
todo -list
```

- Delete todo by id

```
todo -delete <todo-id>
```

- Mark todo as done

```
todo -complete <todo-id>
```
